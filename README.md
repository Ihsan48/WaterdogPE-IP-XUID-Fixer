# WaterdogPE-IP-XUID-Fixer - V1.0.0
<hr />

A PocketMine-MP plugin that fixes the wrong setted Ip and Xuid player-settings in PocketMine-MP caused by Waterdog(PE).

### What does this fix?

##### IP
Before (unfixed): server-ip

Now (fixed): player-ip

<br />

##### XUID
Before (unfixed): ""

Now (fixed): player-xuid
